Alley

No bloat layouts

Tiny footprint

Only 1KB for the layout and reset css. Tested working on Firefox, Safari, Chrome, IE8+

Completely responsive

Use table-cell layouts which adapt to your browser using percentages.

Object-oriented themes

Create reusable themes, keeping your layout and styling separate.

Example html

<div class="row">
    <div class="col w1of3">1of3</div>
    <div class="col w1of3">1of3</div>
    <div class="col w1of3">1of3</div>
</div>
